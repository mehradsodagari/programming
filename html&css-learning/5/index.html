شروع یادگیری HTML

نام نویسنده : مهراد سوداگری

تاریخ انتشار : 1405/02/16

برای ورود به دنیای طراحی سایت ابتدا باید HTML را فرا بگیرید , زیرا با استفاده از ان میتوانید به مرورگر بفهمانید که هر بخش در کدام قسمت از سایت و به چه شکلی نمایش داده شود.

یادگیری جنگو مقدماتی

نام نویسنده : علی بیگدلی

تاریخ انتشار : 1405/01/28

در دنیای توسعه وب ، فریمورک Django یکی ار محبوب ترین ابزار ها برای ساخت وبسایت ها و اپلیکیشن های پیشرفته است. Django با فدرت بالا و انعطاف پذیری منحصر به فردش ، توانسته جایگاه ویژه ای در میان توسعه دهندگان و شرکت های فناوری پیدا کند.

پایتون مقدماتی تا پیشرفته

نام نویسنده : جادی میرمیرانی

تاریخ انتشار : 1405/06/13

پایتون یکی از بهترین زبان های حال حاضر برنامه نویسی است و با توجه به کتابخانه ها و فریمورک های متعددی که دارد توجه بسیاری از برنامه نویس ها را به خودش جلب کرده است.

Git و GitHub

نام نویسنده : جادی میرمیرانی

تاریخ انتشار : 1405/05/30

Git و GitHub فقط ابزار نیستند : زبان مشترک کار تیمی در دنیای توسعه نرم افزارند. اگر برنامه نویسی بلد باشید اما نتوانید با Git و GitHub درست کار کنید ، عملا وارد دنیای پروژه های واقعی ، تیمی و حرفه ای نشده اید.
